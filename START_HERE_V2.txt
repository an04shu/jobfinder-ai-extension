STEP 1: GET FREE API KEY
   → Visit: https://rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch
   → Sign up (use Google/GitHub - it's instant)
   → Click "Subscribe to Test" → Select "Basic (FREE)"
   → Copy your API key

STEP 2: CONFIGURE EXTENSION
   → Open "popup.js" in any text editor (Notepad works!)
   → Find line 10: apiKey: 'YOUR_RAPIDAPI_KEY_HERE'
   → Paste your key between the quotes
   → Save the file

STEP 3: RELOAD & USE
   → Open Chrome: chrome://extensions/
   → Find "JobFinder AI" and click reload button (🔄)
   → Click extension icon → "Fetch Jobs"
   → SEE REAL JOBS! 🎉

"Failed to fetch jobs"
→ Check API key is correct (no spaces)
→ Verify you subscribed to JSearch API
→ Check internet connection

Extension won't reload
→ Go to chrome://extensions/
→ Turn developer mode OFF then ON
→ Remove extension and re-add it

→ Change 'week' to 'month' in popup.js line 73
→ Try a broader search query
