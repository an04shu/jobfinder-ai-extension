⚡ QUICK START (3 Steps):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GENERATE ICONS
   → Open "generate_icons.html" in your browser
   → Click "Generate Icons" button
   → Download all 3 icons and save in this folder

2. LOAD IN CHROME
   → Go to: chrome://extensions/
   → Enable "Developer mode" (top-right toggle)
   → Click "Load unpacked"
   → Select this JustApply folder

3. TEST IT!
   → Click the JobFinder AI icon in Chrome
   → Click "Fetch Jobs"
   → Browse job listings
   → Click "View Job" to apply

❓ TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Extension won't load?
  → Make sure Developer Mode is ON
  → Check all files are in JustApply folder
  → Try reloading from chrome://extensions/

Icons not showing?
  → Use generate_icons.html to create them
  → Make sure they're saved in JustApply folder
  → Names must be exact: icon16.png, icon48.png, icon128.png

Jobs not displaying?
  → Right-click extension icon → "Inspect popup"
  → Check console for errors
  → Make sure popup.js is in the folder
